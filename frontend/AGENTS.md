## WHERE TO LOOK
| Task | Location | Notes |
|------|----------|-------|
| Add API endpoint | `src/service/api/` | Function: `fetchXxx`, use `request` from `@sa/axios` |
| Add Query hook | `src/service/hooks/` | Pattern: `useXxx`, combine with `QUERY_KEYS` |
| Add page | `src/pages/(base)/` | Run `pnpm gen-route` after, export `handle` for menu |
| Add feature module | `src/features/<name>/` | Include: hooks/, store/, components/, index.ts |
| Add global component | `src/components/` | PascalCase.tsx, export from index.ts |
| Add Redux slice | `src/store/` or feature | Use `createAppSlice`, export selectors/actions |
| Add i18n text | `src/locales/langs/` | Keys: `route.xxx`, `page.xxx`, `common.xxx` |
| Add theme token | `src/theme/vars.ts` | CSS var: `--color-name`, UnoCSS shortcut: kebab-case |

## CONVENTIONS
- **Path alias**: `@/*` → `src/*`, `~/*` → root
- **Naming**: Components PascalCase, hooks camelCase with `use` prefix, constants SCREAMING_SNAKE_CASE
- **API**: `fetchXxx` functions, `Api.*` namespace types in `src/service/types/`
- **Query keys**: Centralized in `src/service/keys/`, pattern: `QUERY_KEYS.MODULE.KEY`
- **URLs**: Centralized in `src/service/urls/`, pattern: `MODULE_URLS.ENDPOINT`
- **Routing**: Segment syntax `(base)`/`(blank)`/`_builtin`, dynamic params `[id].tsx`
- **State**: Redux for global sync state, TanStack Query for server state, Zustand for feature-local

## ANTI-PATTERNS
- **NEVER** use `any` - use `unknown`, specific interfaces, or generics
- **NEVER** hardcode API URLs - use `*_URLS` constants
- **NEVER** hardcode Query keys - use `QUERY_KEYS`
- **NEVER** mix npm/yarn - use pnpm only
- **NEVER** edit `auto-imports.d.ts` - generated by script
- **NEVER** skip `pnpm gen-route` after adding pages
